# 🧘 Yoga Website

A beautiful full stack yoga website for everyone (Women, Men, Kids).

---

## 🚀 Features

- 🧘 Yoga for Women, Men, Kids
- 📸 Images with each yoga pose
- 💡 Pro tips for better health
- 🔄 Dynamic data from backend
- 🎯 Category-based filtering (buttons)
- 🌐 Clean and responsive design

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

---

## ⚙️ How to Run

1. Clone this repository  
2. Open terminal in backend folder  

3. Open index.html with Live Server

## 💡 Description

This project helps users learn yoga with proper guidance.  
Each category provides useful tips and images to improve health.
